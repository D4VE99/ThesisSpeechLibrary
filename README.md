# ThesisSpeechLibrary
PT-Text-Preprocessor 🇵🇹
Enhancing Portuguese Text-to-Speech (TTS) via Pre-Synthesis Linguistic Processing
PT-Text-Preprocessor is a lightweight JavaScript library designed to optimize raw text specifically for the Web Speech API. It focuses on improving the prosody, pronunciation, and phonetic accuracy of the Portuguese language during Text-to-Speech synthesis.

🔬 Research Context
In many browser-based TTS engines, the Portuguese language often suffers from incorrect stress placement, poor handling of abbreviations, or unnatural pauses. This library acts as a middleware layer that cleans and transforms text before it reaches the synthesis engine, ensuring a more natural and intelligible output.

Key Features
Phonetic Normalization: Handles common Portuguese abbreviations and symbols.

Punctuation Tuning: Adjusts pauses for better rhythmic flow (prosody).

Character Cleaning: Removes or replaces non-standard characters that confuse the Web Speech API.

Lightweight & Dependency-free: Designed to work directly in the browser.
